# SENG272-HW1

This project is a Java Swing desktop application developed for Lab Experiment 1.

## Project Description

The application collects software project registration information and saves it into a local text file named projects.txt.

## Features

- Java Swing GUI
- Project name input
- Team leader input
- Team size selection
- Project type selection
- Start date input
- Save button with validation
- Clear button
- File I/O using BufferedWriter and FileWriter
- Dynamic record time using Java Date/Time API

## How to Run

1. Open the project in IntelliJ IDEA, Eclipse, or VS Code.
2. Compile the Java files inside the src folder.
3. Run ProjectFormApp.java.

## Output File

Saved project records are appended to projects.txt.
